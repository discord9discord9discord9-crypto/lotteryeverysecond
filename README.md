# Lottery Every Second

A lottery simulator that plays EuroJackpot and Powerball every single second to demonstrate how unlikely winning really is.

## Development

```bash
deno task dev
```

## Production

Run with Docker Compose:

```bash
docker-compose up -d
```

Or with Traefik:

```bash
docker-compose -f docker-compose.traefik.yml up -d
```
